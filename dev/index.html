SymbolicCRN.PolyPolyt — Type
struct PolyPolyt{T}

A polynomial p, its Newton polytope Newtonpolyt (as Polymake big object, with the vertices computed) and the map vertices -> points vertex_point_map (vector storing the position of vertex in polynomial p).

Methods to create a PolyPolyt:

PolyPolyt(p::MPolyElem) # Requires computing the vertices of the Newton polytope.
PolyPolyt(p::MPolyElem, vertices, ver_pt_map) # Loads vertices and ver-to-pt map.
PolyPolyt(p::MPolyElem, name) # Loads them from files generated by save_polypolyt(name::String, pp)

SymbolicCRN.tpoly — Type

Structure of a tpolynomial P. Vector p of coefficients of P, with v[1] != 0 (nonzero independent term) and integer mindeg indicating the smallest degree of P. mindeg can be negative, we have, with n = length(v)-1,

     poly = t^(P.mindeg) * sum( P.p .* [1, t,..., t^n])

SymbolicCRN.Jacobian — Function
Jacobian(R, F)
Jacobian(R, F, vars)

Returns the Jacobian matrix of a one-column matrix of polynomials F with respect to the generators of R indexed by vars. When vars is omitted all the generators of R are used.

Examples

julia> using Nemo

julia> R, vars = PolynomialRing(ZZ, vcat(["k$i" for i in 1:5], ["x$i" for i in 1:4]));

julia> F = [vars[1]*vars[2]*vars[6]-vars[8]; vars[3]*vars[9]+2*vars[7]]
[k1*k2*x1-x3]
[ k3*x4+2*x2]

julia> Jacobian(R, F, 6:9)
[k1*k2  0  -1   0]
[    0  2   0  k3]

julia> Jacobian(R, F)
[k2*x1  k1*x1   0  0  0  k1*k2  0  -1   0]
[    0      0  x4  0  0      0  2   0  k3]

SymbolicCRN.Jacobian_cp — Method
Jacobian_cp(N, Y, E, ls, hs)

Returns Nemo's MatElem paramatrising the Jacobian of the dynamical system associated to a biochemical reaction network. We use "convex parameters", see

rn = @reaction_network begin
    k1,  X2 → X1
    k2,  X1 → X2
end k1 k2

N = netstoichmat(rn)
Y = substoichmat(rn)

E = raysof(cone_positivenullspace(N))
W = conservationlaws(N)

SymbolicCRN.vertex_index — Method
vertex_index(pp, i)

Given a vertex i of a PolyPolyt, returns the position of i in ver-to-pt map. That is, vertex_point_map(pp)[vertex_index(pp, i)] == i. Or more interestingly,

verticesof(Newtonpolytope(pp))[:, vertex_index(pp, i)] == collect(exponent_vectors(pp.p))[:,i]
